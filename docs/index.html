The differentiator

Free by default. AI optional.

Every other AI-email tool wants a subscription. Lazy Me ships with a built-in rule-based sorter, so it works with zero API keys. Add a Gemini or OpenAI key later only if you want AI-written summaries — the pipeline never breaks either way.

How it works

Three steps, then you forget it exists

Step 01 · Fetch

Read the day's Gmail

Pulls that day's messages through the official Gmail API over OAuth — read-only, no password ever stored.

Step 02 · Sort

Tag into categories

A free heuristic sorts by sender and subject into nine buckets. Add an AI key and it writes summaries instead.

Step 03 · Deliver

Text you the digest

One card per category lands in Telegram, each tagged and tappable — including a button to trash the whole tag.

Features

Small tool, thorough about your inbox

CATEGORIES

Nine smart buckets

Rejections, interviews, applications, job alerts, finance, promos, personal, newsletters, and a catch-all.

COST

Zero-key mode

Works fully free with a rule-based sorter. AI providers are a drop-in upgrade, never a requirement.

CONTROL

Trash by tag

One tap moves an entire category to Gmail Trash — recoverable for ~30 days, never a hard delete.

FILTER

Tappable #tags

Each card ends in a hashtag you can tap in Telegram to filter that category across every day.

RANGE

Any date

Re-run a digest for today, yesterday, or any specific date with a single flag.

SCHEDULE

Runs itself

A GitHub Actions cron fires it daily in the cloud — your machine doesn't need to be on.

Quick start

Up and running in five commands

Python 3.9+, a Google account, and a Telegram account is all you need.

# clone & install
git clone https://github.com/J-anubhav/Lazy-Me.git
cd Lazy-Me && pip install -r requirements.txt

# authorize Gmail — one-time, opens a browser
python auth_setup.py

# preview today's digest, sends nothing
python src/main.py --dry-run

# for real, lands in Telegram
python src/main.py
